1. What does this tool do?

This tool lets node operators exit their validators using only their mnemonic (no node access needed).

2. Why do I only need a mnemonic, without node access?

Because the tool can derive validator keys from your mnemonic and sign the exit message locally, node access is unnecessary.

3. How does it integrate with ethdo and beaconcha.in?

The tool uses ethdo to sign exit messages and the beaconcha.in API to broadcast them to the network.

4. Is it secure?

Security is paramount. The tool is open-source, can be locally hosted, and has been designed with audits in mind to ensure trustworthiness.
