Exit Process Guide

  1. Review the FAQ: Before you begin, please carefully read our FAQ for important details.
  2. Set a Separate Withdrawal Address: Ensure that your withdrawal address is different from your node address.
  3. Enter Your Mnemonic: Input your mnemonic phrase as shown in the example below.
  4. Select the Correct Node Address: Choose the correct node address. The interface displays the three most common derivation paths.
  5. Extended Settings:

    If your node address isn't displayed, click the "Extended Settings" message to reveal additional options. In this section, you can manually specify the derivation path.

    The derivation path must start with m/44/60 (or m/44'/60' for hardened derivation). For example, you might use m/44/60/123/0/0 or m/44'/60'/132'/0'/0. Alternatively, you can include a placeholder for the index with a path like m/44'/60'/123'/0'/%d (where %d is replaced by the desired index).


    You can also specify a custom index (the default is 0) if necessary.

  6. Load the Minipool List: The minipool list will load shortly. Note that without a Beaconcha.in API key, this may take a few seconds.
  7. Select Validators to Exit: Choose the minipools/validators you wish to exit, then click the "Sign Exit" button to generate the exit message. A confirmation prompt will appear.
  8. Copy and Broadcast the Exit Message: Click the button to copy your exit message, then follow the provided link to Beaconcha.in to broadcast it.
  9. Submit the Message: Paste the exit message and submit it. The broadcast process may take a few moments; once complete, the final status will display as "COMPLETED".
